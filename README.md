# Установка сервера Xray (VLESS TCP + XHTTP + Hysteria 2)

## Описание

Скрипт `install_xray.sh` автоматически разворачивает мультипротокольный Xray-сервер на портах 443 TCP/UDP с использованием TLS (Let's Encrypt), который включает:

- **VLESS TCP XTLS-Vision** (Порт 443 TCP) — скоростной и стабильный стандартный протокол.
- **VLESS XHTTP** (Порт 443 TCP, локальный проброс) — обход глубокого анализа пакетов (DPI) по HTTP/2 пути `/xhttppath/`.
- **Hysteria 2** (Порт 443 UDP) — сверхбыстрый QUIC-протокол для работы в условиях нестабильного или высокозадержного соединения.
- **Поддержка подписок (Base64)**: Сервер автоматически разворачивает локальный Python-демон подписок. Ссылка вида `https://домен/sub/UUID` возвращает все 3 конфигурации одной строкой для импорта во все популярные клиенты.
- **Автоопределение геолокации**: Скрипт сам запрашивает гео-IP сервера и автоматически добавляет флаг-эмодзи страны в имя ссылки (например, 🇳🇱).
- **Кастомные имена устройств**: Возможность задать понятное имя для каждого профиля при установке (например, `iPhone_Alex`, `Macbook_Pro`).
- **Автоматизация**: Получение и автопродление Let's Encrypt сертификатов через Certbot.
- **Фаервол (UFW)**: Автоматическая настройка безопасных правил для TCP/UDP портов.

## Установка и запуск
#### Примечание: Скрипт использует ufw для открытия портов 443 TCP/UDP, 80 TCP и 22 TCP.
Перед установкой убедитесь в доступности порта 22, чтобы не потерять доступ к серверу:
```bash
ufw allow 22/tcp
ufw enable
```

#### Удаленное выполнение:
```bash
sudo bash <(curl -fsSL "https://raw.githubusercontent.com/mvrvntn/vless-server-install/main/install_xray.sh?v=$(date +%s)")
```

#### Локальное выполнение:
```bash
git clone https://github.com/mvrvntn/vless-server-install/ && cd vless-server-install && sudo bash install_xray.sh
```

## Режимы запуска

### 1. Интерактивный режим
```bash
sudo bash install_xray.sh
```
Скрипт запросит:
- Доменное имя
- Email (для Let's Encrypt)
- Количество устройств
- Кастомное имя для каждого устройства (например, `iphone`, `home-pc`)

После завершения установки используйте:
```bash
generate_client_config
```
Выберите устройство, и утилита выведет все три ссылки (VLESS Vision, VLESS XHTTP, Hysteria 2) с соответствующими эмодзи страны и кастомным именем, а также предложит отобразить QR-код.

### 2. Headless режим (без взаимодействия)
```bash
sudo bash install_xray.sh --headless your.domain.com your@email.com 3 "iphone" "macbook" "pc"
```
Параметры:
1. Домен
2. Email
3. Количество клиентов
4. Список кастомных имён через пробел (если не указаны, будут названы по умолчанию `client_1`, `client_2`...)

## Удаление

Если Xray уже установлен, просто запустите скрипт повторно:
```bash
sudo bash install_xray.sh
```
Скрипт определит предыдущую установку и предложит меню, в котором можно выбрать пункт **Удалить Xray**. При этом:
- Удаляются службы и конфигурации Xray
- Удаляются cron-задачи
- Из брандмауэра удаляются правила для портов 443 (TCP/UDP) и 80 (TCP) (остальные правила UFW не затрагиваются)

## Выходные файлы:

- **Конфигурация сервера**: `/usr/local/etc/xray/config.json`
- **Клиентские конфиги**: `/etc/xray/client_configs/*.json`
- **Генератор ссылок**: `/usr/local/bin/generate_client_config`
- **Сервер подписок (Python)**: `/usr/local/bin/xray_sub_server.py`
- **Служба подписок (Systemd)**: `/etc/systemd/system/xray-sub.service`
- **Логи**: `/var/log/xray/`
